JoJo Stands:
JoJoStands is a mod where we're trying to implement many stands and items from 1-7.

JoJoStands forum post: https://forums.terraria.org/index.php?threads/jojostands.77244/

The JoJoStands Wiki: https://terrariamods.fandom.com/wiki/JoJoStands

The Discord Link: https://discord.gg/EYKBhpK

The mod is attempting to follow an open-source approach to continue the development of the mod.
If there is any content that you feel would be nice to have in the mod and are capable of adding, feel free to clone the mod and submit a pull request in the main Github branch.
That said, the mod is also looking for a new maintainer, i.e. take over the server and repositories and manage updates from now on. If you're interested, join the Discord and reach out!
